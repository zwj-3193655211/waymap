# Waymap · 路线规划地图

> 单文件路线规划地图，给旅游出行、三下乡、研学旅行用。

在地图上拖放站点，画连线把它们串起来，标记每个站点已完成 / 待办，最后**把整段行程导出成一个独立的 HTML 文件**——任何人都能用浏览器直接打开，不需要安装、不需要服务器。

## 功能

- 在高德地图上添加 / 编辑站点
- 连线画多段路线
- 标记每个站点为待办 / 已完成，自带进度统计
- 地点搜索，支持高德 / 卫星 / 深色三种底图切换
- 一键导出 → 单 HTML 文件（团队分享的杀手锏）

## 文件

| 文件 | 说明 |
|---|---|
| `实践行.html` | 主程序，用任何现代浏览器打开即可 |
| `favicon.svg` | 站点图标 |
| `key获取方法.txt` | 如何申请免费的高德 Web 服务 API Key |

## 快速开始

1. 申请一个免费的高德 Key（见 `key获取方法.txt`）

2. 打开 `实践行.html`，按提示粘贴 Key

3. 开始搜索地标、画路线、规划地点任务

## 项目由来

最初是为**三下乡**社会实践做的行程规划工具，但通用性足够强，任何需要按站点串路线的出行场景都能用——研学、徒步、自驾、调研都可以。
